# 50行pcm音频播放器

基于python、pysdl2的音频播放器：

- **开发环境是ubuntu mate 16.04**
- **代码兼容python2、python3**
- **请安装pysdl2支持包**
- **sudo apt install python-sdl2**
- **代码总共不到50行**
- **为在最短的代码下播放音频，目前仅支持pcm音频**
- **后续将支持更多格式**
